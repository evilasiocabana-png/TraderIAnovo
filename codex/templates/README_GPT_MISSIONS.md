# Templates de Missoes GPT

Use estes arquivos quando uma melhoria for pensada no GPT e executada pelo
Codex:

- `GPT_IMPROVEMENT_MISSION_TEMPLATE.md`: template completo de missao.
- `MISSION_TEMPLATE.md`: template generico legado.
- `ACCEPTANCE_TEMPLATE.md`: criterios de aceite basicos.
- `EXECUTION_REPORT_TEMPLATE.md`: relatorio de execucao.
- `ERROR_REPORT_TEMPLATE.md`: relatorio de falha.

O guia de uso fica em:

```text
docs/GPT_MISSION_AUTHORING_GUIDE.md
```

Regra principal: o GPT deve gerar pacote de missao, nao instrucao solta.
